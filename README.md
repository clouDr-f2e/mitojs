## 概要

一款轻量级的收集页面的用户点击行为、路由跳转、接口报错、代码报错、并上报服务端的SDK

## 为什么使用mitojs
* 🔨&nbsp;&nbsp;监听请求错误、console、路由跳转、代码报错、click、资源加载错误
* 🏅&nbsp;&nbsp;自定义上报错误
* 🤙&nbsp;&nbsp;`mito.min.js`只需16k
* 👌&nbsp;&nbsp;持续迭代与更新
